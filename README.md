# 📊 Dashboard de Marketing com IA

Dashboard interativo para análise de campanhas de mídia paga com integração de Inteligência Artificial (ChatGPT).

## 🚀 Funcionalidades

- Upload de CSV com dados de campanhas
- Cálculo automático de métricas:
  - CTR
  - CPC
  - CPA
  - ROI
- Filtros dinâmicos por canal
- Visualização com gráficos interativos
- Ranking de campanhas
- Chat com IA para análise estratégica

## 🤖 Inteligência Artificial

O projeto utiliza a API da OpenAI (ChatGPT) para:

- Analisar campanhas automaticamente
- Gerar insights estratégicos
- Responder perguntas sobre os dados


## ⚙️ Tecnologias utilizadas

- Python
- Streamlit
- Pandas
- Plotly
- OpenAI API
